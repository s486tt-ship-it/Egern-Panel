# Egern 机场订阅流量面板

一个用于 Egern 的模块，可以在面板上实时显示机场订阅的流量使用情况和到期时间。

## 版本信息
**当前版本**: v1.0.3

## 功能介绍
- **流量统计**: 自动解析并显示已用、总计及剩余流量。
- **可视化进度条**: 直观展示流量消耗比例。
- **到期提醒**: 实时计算订阅剩余天数，防止断连。
- **多机场支持**: 默认支持配置两个不同的机场订阅。
- **参数化配置**: 支持在模块设置中直接修改机场名称和订阅链接。
- **动态加载**: 修复了脚本语法错误和面板变量替换失效的问题。

## 安装与使用
1. **复制订阅地址**: 
   ```text
   https://raw.githubusercontent.com/s486tt-ship-it/Egern-Panel/main/panel/Subscription%20Panel/AirportSub.yaml
   ```
2. **添加到 Egern**: 
   在 Egern 的 **工具** -> **模块** -> **添加模块** 中粘贴上述地址。
3. **配置参数**: 
   点击模块进入设置，填写您的机场名称和订阅链接。
