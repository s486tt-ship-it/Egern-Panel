# 机场订阅流量信息面板

这是一个为 Egern 设计的精美面板插件，用于实时监控一个或多个机场订阅的流量消耗和到期时间。

## 🌟 功能特性

- **多订阅支持**：支持同时管理多个机场订阅链接
- **流量统计**：实时显示已用、剩余、总流量及使用百分比（GB）
- **到期提醒**：自动解析订阅过期时间并显示剩余天数
- **便捷配置**：支持在 Egern 模块参数界面直接添加和修改订阅链接
- **实时更新**：每小时自动刷新一次流量数据
- **优雅展示**：使用emoji图标美化展示效果

## 🛠️ 安装方法

在 Egern 的"模块"页面添加以下 **Raw** 链接：

```
https://raw.githubusercontent.com/s486tt-ship-it/Egern-Panel/main/Egern-Panel/Subscription-Template/airport_panel.yaml
```

## ⚙️ 参数配置

### 单个订阅链接

1. 启用模块后，进入模块详情页
2. 点击 **"编辑模板参数"**
3. 在 `SUBSCRIPTION_URLS` 框中填入你的订阅链接

示例：
```
https://sub01.sh-cloudflare.sbs:8443/api/v1/client/subscribe?token=571791988e327f4615fcaefff79c2f23
```

### 多个订阅链接

使用分号 `;` 分隔多个链接：

```
https://sub01.example.com/api/v1/subscribe?token=xxx;https://sub02.example.com/api/v1/subscribe?token=yyy
```

## 📊 显示信息说明

面板会显示以下信息：

- **✈️ 机场号**：订阅序号
- **已用流量**：上传 + 下载的总流量（GB）
- **剩余流量**：总流量 - 已用流量（GB）
- **总流量**：订阅包含的总流量（GB）
- **使用率**：已用流量/总流量的百分比
- **到期时间**：订阅到期日期和剩余天数

## 🔗 常见问题

### Q: 为什么显示"无法获取流量信息"？
A: 这可能是因为：
- 订阅链接不正确或已失效
- 服务器未返回标准的 `subscription-userinfo` 头部
- 网络连接问题

### Q: 可以添加多少个订阅？
A: 理论上没有限制，但建议不超过5个以保证显示效果

### Q: 更新频率是多少？
A: 默认每小时更新一次（3600秒），可在配置文件中修改 `update_interval` 参数
