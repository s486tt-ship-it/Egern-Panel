# 机场订阅详细 — Egern 2.16 流量面板 v2.2

> 在 Egern 面板中查看机场订阅的**已用流量**、**剩余流量**和**到期时间**。  
> 独创的「毫秒级缓存回退 + 伪装主动探测」机制，杜绝面板白屏挂起！

---

## 文件说明

| 文件 | 用途 |
|------|------|
| `SubPanel.yaml` | Egern 模块配置（含模板参数定义） |
| `sub_panel.js` | 面板脚本，主动且安全地探测最新流量展示（超时自动降级读缓存） |
| `sub_store.js` | 备用的被动拦截脚本，更新订阅时采集 |

---

## 安装方式

### 远程模块订阅（推荐）

在 Egern 中操作：  
**模块 → + → 输入 URL**，填入下方链接：

```
https://raw.githubusercontent.com/s486tt-ship-it/Egern-Panel/main/Egern-Panel/%E8%87%AA%E5%88%B6-Panel/%E6%9C%BA%E5%9C%BA%E8%AE%A2%E9%98%85%E8%AF%A6%E7%BB%86/SubPanel.yaml
```

---

## 配置模板参数（必填！）

安装模块后，进入 **编辑模板参数**，由于部分防爬虫的机场必须要带有 `token` 才能取回信息，**请务必填入完整订阅 URL**：

| 参数名 | 说明 | 示例 |
|--------|------|------|
| `AIRPORT1_NAME` | 机场1的显示别名 | `我的主力机场` |
| `AIRPORT1_URL` | 机场1的完整订阅链接（必须填） | `https://example.com/sub?token=xxx` |
| `AIRPORT2_NAME` | 机场2的显示别名（可留空） | `备用机场` |
| `AIRPORT2_URL` | 机场2的订阅链接（可留空） | `https://other.com/sub?token=yyy` |
| `AIRPORT3_NAME` | 机场3的显示别名（可留空） | — |
| `AIRPORT3_URL` | 机场3的订阅链接（可留空） | — |

## 数据触发及使用方法（必看）

在 v2.2 中，你只需要：
1. 配置好模板参数的「名称」与「完整订阅URL」
2. 前往 Egern **首页 → 面板** 找到此面板
3. 点击右上角的 **↻ 刷新**
4. 面板会以 `Clash` 的身份帮你秘密探测流量并光速返回结果，如有网络延时，会自动降级显示上次扫描的旧缓存避免白屏！

> **兼容性说明：**
> 如果你的机场不支持 `Subscription-Userinfo` 头部规范（例如使用 WHMCS 作为源后端的 `app.mitce.net`），面板会友好提示“该机场未在响应头提供流量信息”，而不是直接报错或死机。
