# 机场订阅详细 — Egern 2.16 流量面板 v2.3

> 在 Egern 面板中查看机场订阅的**已用流量**、**剩余流量**和**到期时间**。  
> 独创的「毫秒级防死机机制 + JS原生界面绑定」，从此告别白屏！

---

## v2.3 的重大改进（必读）

针对部分用户反馈的「空白页面」和由于 YAML `compat_arguments` 兼容性引发的无法配置问题，v2.3 彻底废弃了从 YAML 传入配置的间接方案，**直接使用 JS 原生配置引擎 (`#!arguments`)**。这带来了两点革命性提升：
1. **参数配置永不丢失**：长按面板直接编辑，兼容性完美。
2. **两秒防爆死机制**：针对故意拉黑抓取、死等不带 `Userinfo` 头的流氓机场，主动切断请求并强制展示本地历史流量记录，拒绝白屏等待。

在 Egern 中操作：

```

```

## 如何配置及使用面板

1. 添加模块后，回到 **Egern 首页 → 面板** 中找到本面板。
2. **长按此面板**，选择 **编辑参数 (Edit Arguments)**。
3. 填入你的机场信息（**`AIRPORT1_URL` 是必填，否则查不到流量**）：
   - `AIRPORT1_NAME`: 我的机场
   - `AIRPORT1_URL`: `https://example.com/sub?token=xxx`
4. 保存参数后，直接点击面板右上角的 **↻ 刷新** 即可生效！

*(注：如果在长按面板找不到参数编辑，也可以去大菜单的 脚本 → 编辑参数 里修改。)*
