# 机场订阅详细 — Egern 2.16 流量面板 v2.1

> 在 Egern 面板中查看机场订阅的**已用流量**、**剩余流量**和**到期时间**。  
> 支持最多 3 个自定义机场名称。

---

## 文件说明

| 文件 | 用途 |
|------|------|
| `SubPanel.yaml` | Egern 模块配置（含模板参数定义） |
| `sub_panel.js` | 面板脚本，读取本地缓存数据并展示 |
| `sub_store.js` | 采集脚本，在你更新机场订阅时自动拦截并收集数据 |

---

## 安装方式

### 远程模块订阅（推荐）

在 Egern 中操作：  
**模块 → + → 输入 URL**，填入下方链接：

```
https://raw.githubusercontent.com/s486tt-ship-it/Egern-Panel/main/Egern-Panel/%E8%87%AA%E5%88%B6-Panel/%E6%9C%BA%E5%9C%BA%E8%AE%A2%E9%98%85%E8%AF%A6%E7%BB%86/SubPanel.yaml
```

---

## 配置模板参数（机场重命名）

安装模块后，进入 **编辑模板参数**，你可以为你的机场配置自定义名称：

| 参数名 | 说明 | 示例 |
|--------|------|------|
| `AIRPORT1_NAME` | 机场1的显示别名 | `我的主力机场` |
| `AIRPORT1_URL` | 机场1的订阅链接（仅用于对应识别，不会发请求） | `https://example.com/sub?token=xxx` |
| `AIRPORT2_NAME` | 机场2的显示别名（可留空） | `备用机场` |
| `AIRPORT2_URL` | 机场2的订阅链接（可留空） | `https://other.com/sub?token=yyy` |
| `AIRPORT3_NAME` | 机场3的显示别名（可留空） | — |
| `AIRPORT3_URL` | 机场3的订阅链接（可留空） | — |

> 提示：面板脚本会根据你这里的 `URL` 的前缀来判断应该将 `NAME` 赋给哪个订阅数据。如果不填，面板依然会默认显示机场原来的域名。

---

## 数据触发及使用方法（必看）

1. 在 Egern 中进入 **订阅** 页面
2. 找到你的机场订阅，点击 **更新** 按钮
3. 更新时，底层的 `sub_store.js` 会自动拦截响应头，分析并持久化流量数据
4. 随后，前往 Egern **首页 → 面板**
5. 点击刷新，即可看到最新的流量信息

## 常见问题

**Q：面板显示为空白或者没有任何内容？**  
A：检查 Egern 模块界面的 `sub_panel.js` 是否加载成功。如果是刚修改的参数，你可以尝试返回首页点击**重启**来刷新脚本引擎作用域。

**Q：面板显示"暂无订阅数据"？**  
A：说明之前没有成功拦截到数据。请去**订阅**列表手动点击你机场订阅的"更新"按钮。

**Q：为什么我点击更新订阅后仍然没有数据？**  
A：部分机场的订阅链接没有遵循标准，或者不会返回 `subscription-userinfo` HTTP 头。你可以把订阅链接发到浏览器里打开，按 F12 看看 Network Response Header 里是否有流量信息，如果没有，说明该机场不支持此功能。
